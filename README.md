# Advanced VLSI Design Projects

This repository contains the hardware design projects for the Advanced VLSI Design (Spring 2026) course.

## Projects

### 1. Final Project: [2D DCT Image Compression Accelerator](Final-DCT/README.md)
A highly parameterized, multi-variant 2D Discrete Cosine Transform (DCT) hardware accelerator implemented in SystemVerilog. Optimized for JPEG image compression, this project features 4 distinct architectural variants (Baseline, Pipelined, Parallel, Pipelined+Parallel) exploring the Pareto frontier of area vs. throughput.

### 2. Midterm Project: [FIR Filter Design](Project-FIR/README.md)
An FPGA implementation of a Digital Finite Impulse Response (FIR) Filter.
